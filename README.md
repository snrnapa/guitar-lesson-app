

## 全体像の要件
- ギターの練習をするアプリ
- アプリ上で音声とコードが流れるので、実際にギターを弾いて、それをスマホが音声を拾って、できているかどうかを判断するアプリ
- いろいろな曲、というよりかは、まず1曲を完遂して、成功体験を重ねて、その先はスクールなどにいざなうものです。
- ターゲットは初級者、中級者なりたて
- コードを完コピではなくて、アドリブソロの練習を行うものです。
  - 一旦はそこは難しいので完コピで作ってしまってよい
  - 共通としてバッキングは枯れ葉という曲のコードをそのままもってきます
- 難易度は選べて、それによってアドリブの難易度が変化する
  - 初級:１音でアドリブ
  - 中級:5音でアドリブ
  - 上級:10音でアドリブ
